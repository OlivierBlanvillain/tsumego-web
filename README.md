### Initial set up

Install Ruby & bundler, on Debian/Ubuntu:

	sudo apt-get install ruby
	sudo gem install bundler

Initialize git modules in project directory:

	git submodule init

Update submodules, get Ruby dependencies, and build project:

	make

### Updates

Easy, rerun `make`!
